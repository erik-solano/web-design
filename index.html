Text Styles

A text style is a named collection of text settings that controls the appearance of text, such as font, line spacing, justification, and color. For now we will go over the different headings that can be used in CSS.

Here are some headings that are common in CSS.
H1: Main page heading

  • Font-weight: 700 (bold)
  • Font-size: 26px
  • Font-family: Open Sans
H2: Subheading

  • Font-weight: 500
  • Font-size: 18px
  • Font-family: Rubik
P: Paragraph text

  • Font-weight: 400 (regular)
  • Font-size: 14px
  • Font-family: Merriweather
